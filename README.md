# Gear Position Display PCB

This project focuses on the design and development of a custom PCB for a gear position display system. The system detects the current gear position using sensor inputs and displays it in real-time using a digital interface.

## 🚀 Features
- Real-time gear position detection
- Microcontroller-based processing (Arduino / embedded MCU)
- Digital display interface (7-segment / OLED / LCD)
- Sensor input support (hall sensor / switch-based detection)
- Compact and efficient PCB layout
- Automotive-oriented design approach

## 🧠 System Overview
The system reads inputs from gear position sensors (such as Hall effect sensors or mechanical switches). Based on the input combination, the microcontroller determines the current gear and updates the display accordingly.

## 🔧 Hardware Components
- Microcontroller (Arduino Nano / ATmega / ESP32)
- Gear position sensors (Hall effect sensors or switches)
- Display module (7-segment / LCD / OLED)
- Voltage regulator (for stable power supply)
- Signal conditioning components (resistors, capacitors)
- Connectors for sensor and display interfacing
